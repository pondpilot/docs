## The Ecosystem

<CardGrid>
  <Card title="PondPilot" icon="rocket">
    The main data exploration app. Analyze local and remote data with SQL, powered by DuckDB-WASM. Zero setup, 100% client-side.

    [Learn more ‚Üí](/pondpilot/getting-started/)

  </Card>
  <Card title="FlowScope" icon="random">
    SQL lineage engine that analyzes queries to produce detailed lineage graphs showing how tables and columns flow through transformations.

    [Learn more ‚Üí](/flowscope/getting-started/)

  </Card>
  <Card title="Widget" icon="puzzle">
    Transform static SQL code blocks into interactive, browser-based SQL editors. Perfect for documentation and tutorials.

    [Learn more ‚Üí](/widget/getting-started/)

  </Card>
  <Card title="Proxy" icon="setting">
    Multi-tenant data proxy with per-user isolated DuckDB instances. Query PostgreSQL, MySQL, and SQLite via Flight SQL.

    [Learn more ‚Üí](/proxy/getting-started/)

  </Card>
